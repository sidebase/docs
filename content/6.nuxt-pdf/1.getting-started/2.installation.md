# Installation

To install the module begin by installing our package
::code-group
```bash [npm]
npm install -D @sidebase/nuxt-pdf
```
```bash [yarn]
yarn add --dev @sidebase/nuxt-pdf
```
```bash [pnpm]
pnpm i -D @sidebase/nuxt-pdf
```
::

`nuxt-pdf` only requires a Nuxt 3 app to run.
